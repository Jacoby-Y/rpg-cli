# rpg-cli

```
clear / c                        : Clear terminal
quit / q                         : Quit game / save to database
fight / f                        : Fight on your current floor
stats / st                       : Display all player info
autof / af <min health>          : Auto fight until your hp meets <min health>
move / m <number / max>          : Move to a new floor on the level
levelup / lup <stat> {level ups} : Level up a stat for exp
upgrade / up <item> {upgrades}   : Upgrade an item for cash
```
